Release Notes for STSW-BLUEMIC-1

Copyright 2017 STMicroelectronics

 

Audio and inertial data streaming via Bluetooth® low energy using SPBTLE-1S, a very low power application module based on BlueNRG-1.

STSW-BLUEMIC-1 software features:
  • Complete SW package supporting SPBTLE-1S module, based on BlueNRG-1 Bluetooth® low energy system on-chip.
  • Bluetooth® v4.2 compliant: it supports master and slave modes, multiple roles supported simultaneously.
  • Digital audio signal acquisition and compression (ITU-T G.726 ADPCM).
  • Voice streaming over Bluetooth® low energy using a vendor-specific profile available in the package as a binary library.
  • Motion sensor acquisition, 3-axes accelerometer and 3-axes gyroscope data streaming via Bluetooth® low energy.
  • Data transmission to ST BlueMS app (v3.0.0 or higher), available for Android™ and iOS™.
  • Bluetooth® low energy stack binary library and APIs, events callbacks interface; BlueNRG-1 CMSIS files and peripherals drivers.
  • Complete board support package to build application using digital MEMS microphone (MP34DT04-C1) and motion sensor (LSM6DSL).
